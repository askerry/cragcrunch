cragcrunch
========

playing around with climbing data

will inlude:
1) scrapy-based mpscraper for scraping full contents of mountainproject.com
2) SQL databases storing MP data
3) interface for interacting with climbing data and visualizing information about areas (e.g. side by side comparisons of climbing destinations)
4) climb recommendation system that will suggest climbs appropriate for user based on previous ticks/comments/ratings
